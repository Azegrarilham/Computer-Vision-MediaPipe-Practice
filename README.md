# OpenCV & MediaPipe Practice

This repository contains small scripts, experiments, and prototypes created while
learning computer vision concepts using **OpenCV** and **MediaPipe**.

The goal of this repository is learning and experimentation rather than delivering
a polished or production-ready application.

---

## Contents

- **OpenCV basics**
  - Image loading and display
  - Video and webcam capture
  - Drawing shapes and text
- **MediaPipe**
  - Hand detection
  - Face detection
  - Pose estimation
- **Experiments**
  - Camera tests
  - Prototype ideas
  - Trial-and-error scripts
